Load Balancing
Contribute Performance Data

Sandbox is built using the jsdelivr CDN, which uses real user performance data (also known as RUM) to make its routing decisions. This data is gathered from hundreds of websites and is used in jsdelivr's load balancing algorithm to make accurate decisions based on real time performance metrics. The guys at jsdelivr have built a great service and given it to us totally free, which is why we'd like to give you the chance to help them out by contributing to their RUM.




For more information about what information is reported and jsdelivr's privacy policy, check out their docs

Aliasing

Aliasing allows you to easily define shorthand query strings that are tied to your favorite asset combinations. To link an alias to a series of resource queries, simply define your alias and type each library query you wish to include using the standard syntax: 

libarary @version : [files]

If you wish to include multiple libraries in your alias, simply separate them with the unary '&'
jquery @1.11.1 : [jquery.js] & underscorejs

Aliasing will allow you to quickly load your favorite libararies with a couple of keystrokes. The following two function calls are equivalent: 
    
    // if window.Sandbox.aliases = {'$': 'jquery @1.11.1 : [jquery.js]'}
    
    __('$');
    
    __('jquery @1.11.1 : [jquery.js]');
You may additionally specify a special alias $DEFAULT which will be used as an argument if the Sandbox constructor is called without any query strings.

For more information about aliasing, check out the Sandbox.js docs
